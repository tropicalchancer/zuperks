# Zuperks - Product Overview

Zuperks is a platform for members of the Zupass community to access and share exclusive perks and offers. The application allows users to:

1. Browse available perks categorized as either "traditional" (company offers) or "community" (peer-to-peer offers)
2. View detailed information about each perk including redemption steps and codes
3. Propose new perks for the community
4. Authenticate using Zupass credentials

## Key Features

- **Authentication**: Secure access via Zupass verification
- **Perk Discovery**: Browse and filter perks by category
- **Perk Details**: View comprehensive information about each offer
- **Perk Proposal**: Community members can submit new perk ideas

## Target Users

The platform is designed for members of various Zu communities including Zuzalu, ZuConnect, Vitalia, AgoraCore, ZuArchitects, ZuVillageGeorgia, and EdgeEsmeralda.

## Business Logic

- Users must authenticate with valid Zupass credentials to access the platform
- Perks are categorized as either "traditional" (business offers) or "community" (peer-to-peer)
- Each perk contains metadata including name, description, tag, offer details, redemption steps, and codes
- Users can propose new perks which are displayed in a separate proposals section
